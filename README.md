# Bank_Loan_Data_Analysis_Dashboard_PowerBI
📌 Project Overview

The Bank Loan Analysis Dashboard is an interactive Power BI project designed to analyze loan applications, funded amounts, repayments, borrower profiles, and loan performance. The dashboard helps financial institutions monitor loan distribution, assess risk, identify repayment trends, and make data-driven lending decisions.

This project transforms raw loan data into meaningful insights through dynamic visualizations, KPI tracking, and advanced filtering capabilities.

🎯 Project Objectives
Analyze overall loan application trends.
Monitor funded amounts and received payments.
Evaluate loan performance based on loan status.
Identify high-risk and low-risk borrower segments.
Track loan distribution across states, purposes, and employment lengths.
Support decision-making through interactive dashboards.

📌 Key Insights
Identified the percentage of good and bad loans.
Analyzed repayment behavior across different borrower categories.
Tracked funding and collection efficiency.
Evaluated geographical trends in loan approvals and defaults.
Discovered factors influencing loan performance and risk.
🚀 Business Impact

This dashboard enables banks and financial institutions to:

Improve loan approval strategies.
Reduce default risks.
Optimize lending operations.
Enhance portfolio monitoring.
Support data-driven financial decisions.
